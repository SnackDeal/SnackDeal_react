## 현재 상태

별도로 확정된 명세 문서(기획/DB 설계 문서 등)가 아직 이 저장소에 반입되지 않아, 각 `_개요.md`는
현재 코드(`src/types/`, `src/mocks/`, `src/stores/`, `src/pages/`)를 근거로 역추적해서 작성한
**작업용 개요**입니다. 실제 확정 명세(SQL DDL, 기획서 등)가 들어오면 해당 문서로 교체하세요.

각 개요 문서 상단에 이 사실이 명시돼 있고, 문서화되지 않아 추측이 필요했던 부분은 "TODO / 확인
필요" 섹션에 모아뒀습니다. 새로운 화면/로직을 구현할 때 해당 항목을 먼저 팀에 확인하세요.
